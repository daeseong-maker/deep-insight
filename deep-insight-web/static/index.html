Deep Insight

리포팅 등의 다양한 에이전트 응용 개발 위한 커스텀 에이전트 구축 플랫폼

Built on Strands Agent & Amazon Bedrock AgentCore

사용 가이드

▾
  1. 데이터 파일(CSV, Excel 등)과 컬럼 정의 파일(선택)을 업로드합니다
  2. 분석 프롬프트를 입력합니다
  3. 분석 계획이 표시되면 승인하거나 수정을 요청합니다
  4. 에이전트가 분석을 완료할 때까지 기다립니다 (수 분 소요)
  5. Reports 섹션에서 생성된 보고서를 다운로드합니다
⏱ 보고서 생성은 15분 이상 소요될 수 있습니다. 샘플 데이터의 경우 약 15분 정도 걸립니다.
