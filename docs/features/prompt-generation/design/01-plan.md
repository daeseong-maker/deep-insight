# Plan

## Background
아래는 현재 Web-UI Page 의 현재 UI 입니다. 
이것은 Web Page의 기본 샘플 data file 및 컬럼 정의의 json 파일에 기반해서, 사용자가 프롬프트를 쉽계 클릭해서 사용하기 위함 입니다.

```
분석 UI
프롬프트 
- 데이터에서 비즈니스 성장 기회를 발굴해줘: 숨겨진 고객 패턴과 세그먼트를 발견하고, 수익 최적화 방안을 제시하며, 마케팅과 운영 효율성을 높일 수 있는 개선점을 찾고, 다음 달 매출을 크게 늘릴 수 있는 실행 가능한 전략 3가지를 우선순위와 기대 효과를 포함해서 제안해줘
예시를 클릭하면 자동 입력됩니다
간단데이터의 핵심 지표를 요약해줘
간단총 매출액을 계산해줘
상세데이터에서 비즈니스 성장 기회를 발굴해줘: 숨겨진 고객 패턴과 세그먼트를 발견하고, 수익 최적화 방안을 제시하며, 마케팅과 운영 효율성을 높일 수 있는 개선점을 찾고, 다음 달 매출을 크게 늘릴 수 있는 실행 가능한 전략 3가지를 우선순위와 기대 효과를 포함해서 제안해줘
```
사용자가 본인의 데이터 파일을 업로드 하게 되면, 직접 프롬프트를 사용하게 됩니다. 
제공된 데이터 파일 및 컬럼 정의의 JSON 파일을 기반으로 프롬트트 샘플을 자동으로 생성해주고,
사용자가 필요하면 수정해서 사용을 하는 것이 필요 합니다.


## Goals
/home/ubuntu/sample-deep-insight/docs/features/json-generation 를 통해서 생성된 JSON 파일을 기반으로
프롬프트 샘플을 생성을 해주고, 사용자가 클릭을 하게 해줍니다. 
프롬프트 샘플은 복잡도에 따라서, 비즈니스 관점으로 간단, 중간, 복잡의 3가지를 만들면 좋겠습니다.
기존의 UI 처럼 클릭하면, Text Area 에 추가가 되고, 사용자가 필요하면 수정을 합니다. 이후에 클릭하면 프롬프트가 반영이 됩니다.
Json-generation 의 UI 처럼 Prompt 생성하기 버튼을 눌어 주면 좋을 것 같습니다.

구현 코드는 코드의 패턴을 비슷하게 가지고 가기 위해서, Json-generation 의 코드를 가능하면 재사용하세요.
