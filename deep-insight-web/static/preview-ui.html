QUICK START GUIDE

▾
🌐 Chrome browser is recommended
🔒 Uploaded data and generated reports are stored in your S3 bucket. The frontend is restricted to internal company IP ranges, and the backend runs in a private VPC. Your data is securely managed.
  1. 📂Upload your data file (CSV, Excel, etc.) and column definitions
  2. ✏️Enter an analysis prompt describing what you want to analyze
  3. 📋Review the analysis plan when prompted and approve or request changes
  4. ⏳Wait for the agents to complete the analysis
  5. 📥Download the generated reports from the Reports section
💡 Column definitions are optional but recommended. Column names may have domain-specific meanings (e.g., sales = net vs gross), and you can define formulas to apply during analysis.
⏱ Elapsed time varies by prompt complexity, data size, and LLM server load. For sample data, it may take approximately 15-30 minutes.
