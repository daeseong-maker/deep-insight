## ② 오프닝 & 훅

**핵심 원칙**: 관객이 영상을 넘길지 결정하는 시간 = 평균 6초. 첫 씬은 배경 설명이 아니라 **갈등·충격·미스터리**로 시작한다.

| 방식 | 설명 | 교차검증 |
|------|------|---------|
| 결말 선배치 | 결말·죽음을 먼저 보여주고 역추적 진행. "왜?"가 즉각 서스펜스로 작동 | ✅ #3·#6·#9(역삼각 구조와 동일 원리) |
| 충돌 선배치 | 갈등 폭발 장면 먼저 → 배경 보충. 선형 배경 설명 전면 금지 | ✅ #4·#8·#9·#11·#13 — 5편 공통 |
| 극단적 신체 행동 오프닝 | 인물 간 신체 충돌 또는 극단 행동으로 성격 즉각 제시. 설명 없이 행동으로만 | ✅ #4·#8 |
| 미해결 미스터리 즉각 투입 | 1화 1씬에 실종·죽음·반전 제시 | ✅ #1·#3 |
| 회귀·환생 선언식 오프닝 | 1대사에서 즉시 이세계/시간역행 선언 + 전생 갈등 압축 제시. 충돌 선배치 원칙의 장르별 적용 | ✅ #4 원칙 적용·🔶 #13 특화 기술 |

> **훅 부재 실패 패턴**: 나레이션 자기소개, 배경 설명 씬, 일상 묘사 3씬 이상 = ⛔ 높음

## ③ 에피소드 구조 & 클리프행어

**핵심 원칙**: 매 화는 "새 질문 제기 + 불완전 해소 + 다음 질문 암시" 사이클로 종결한다.

### 화 내부 구조 유형

| 구조 | 설명 | 교차검증 |
|------|------|---------|
| 90초 3막 모듈 | 0~15초 충돌 / 16~45초 전개 / 46~90초 감정 클라이맥스 | ✅ #4·#8(단집 4포인트)·#13(위기→반전) — 3편 공통 |
| 충돌-클라이맥스-서스펜스 3단 | 도입부 충돌 → 중간 고조 → 결말 서스펜스 잔류 | ✅ #6 |
| 위기→반전 모듈 | 억압·위기 + 역전·반격 사이클, 화당 최소 1회 | ✅ #4·#13 |
| 역삼각 구조 | 고조 먼저 → 배경은 이후 보충. 결말 선배치·충돌 선배치의 동일 원리 | ✅ #3·#4·#6·#9 — 4편 교차 (**🔶→✅ 승급**) |

### 전체 시리즈 구조 유형

| 구조 유형 | 특징 | 적합 장르 |
|-----------|------|----------|
| 인과선형 | 사건이 시간 순서로 연결 | 역전·성장극 |
| 연쇄 고리 구조 | 각 화 = 하나의 미션/의문 고리, 고리 간 연결로 전체 서사 | 서스펜스 |
| 주부교직 쌍선 | 주선 + 보조선 병행, 보조선이 주선 전환점 강화 | 로맨스·혼인극 |
| 결말 선배치 + 역추적 | 결말 먼저, 이후 역순 전개 | 서스펜스 |
| 블록 독립형 | 각 화 독립 완결, 공통 세계관·인물로 연결 | 치유·옴니버스 |

> 블록 독립형은 ✅ #3·#7·#8·#9·#12·#13 — **6편 교차검증** (**🔶→✅ 승급**, 사실상 전 장르 적용 가능)

### 클리프행어 체크포인트

- 위치: 화 최후 5~10초 이내 ✅ **#2·#7·#9·#13 — 4편 공통** (**🔶 #9·#13 → ✅ 승급**)
- 형태: 미해결 질문 / 충격 정보 투입 / 감정 절정 중단 / 새 인물·사건 등장
- 연속 3화 이상 클리프행어 없음 = ⚠️ 중간 경고

## ④ 반전 & 복선

**핵심 원칙**: 반전의 가치는 **수**가 아니라 **인과**다. 복선 없는 반전 1개 = 복선 있는 반전 0개.

### 복선 설계 기준

| 기준 | 설명 | 교차검증 |
|------|------|---------|
| 복선 필수 원칙 | 모든 반전은 기존 대본 내 단서에서 도출 가능해야 함 | ✅ #1·#2·#4·#6·#7·**#10** — 6편 공통 (**#10 추가 확인**) |
| A/B급 복선 이중 배치 | A급 명시 복선(주선) + B급 숨김 복선(소품·대화 공백). 반복 시청 유도 | ✅ #3 |
| 숨김 복선 구조 | 핵심 정보만 노출, 나머지는 관객이 채우는 여백 설계 | ✅ #1·#4 |
| 소품(Symbol Props) 복선 | 구체 소품이 서스펜스 추진·복선 역할 수행. 16부 기준 6종 이상 | ✅ #1·**#15(소품 서사 기능화)** — **#15 추가 확인** |

### 반전 설계 기준

| 기준 | 설명 | 교차검증 |
|------|------|---------|
| 3단계 반전 구조 | 오독 → 재오독 → 진실 (1사건당 3회 반전 = 서스펜스 적정) | ✅ #1·#10·**#12** — **#12 추가 확인** |
| 다단 반전 | 사건 반전 + 인물 성격·감정 방향 반전 병행 → 장기 몰입 | ✅ #1·#10·#12 (**🔶→✅ 완전 승급**) |
| 제시→전복 사이클 | 정보 제시 직후 뒤집는 전복 설계. 화당 최소 1사이클 | ✅ #7·**#9·#11** — **3편 교차** (**#9·#11 추가 확인**) |
| 확정성 쾌감 구조 | 결말은 예측 가능(주인공 필승), 과정(반전 경로)은 매화 변화 | 🔶 #13 (∴ #3 논리 폐환과 부분 정합. 2편 이하로 유지) |

> **뒤늦은 새 정보 반전** = ⛔ 높음 (#1·#4·#6·#10 — 4편 공통)

## ⑤ 인물 설계

**핵심 원칙**: 숏폼 인물은 핵심 3인 내로 집중하고, 주인공만은 반드시 입체적·성장선을 갖는다.

### 주인공 설계

| 기준 | 설명 | 교차검증 |
|------|------|---------|
| 극적 대비 주인공 | 사회 약자·소인물 출발 → 역전. 시작·최종 지위 낙차 클수록 쾌감 비례 | ✅ #4·**#13** — **#13 추가 확인** |
| 3단계 실패 → 성장 원칙 | 핵심 실패 최소 3번 → 점층 성장. 단일 사건 즉각 변화 = ⚠️ | ✅ #4·#6·**#13(위기→반전 3사이클)** |
| 양면 설계 | 동일 인물이 다른 장면에서 상반된 면 표현. 단면 캐릭터 = 도구 인물 경고 | ✅ #4·#6 |
| 주인공 내부 시점 이중 활용 | 주인공 내면 독백으로 동기 직달 + 적대자 계획 선공개 → "알면서 보는" 긴장 | 🔶 #13 (∴ #1 복수 시점 서술과 부분 정합) |
| 자기 응시 장면 | 인물 각성·전환 전 내면 성찰 장면 필수. 외부 사건만으로는 설득력 저하 | 🔶 **#11·#12 — 그룹 내 2편 확인** (Tier 2 특히 혼인극·로맨스) |

## ⑧ 장르별 심화 기준

### 🔍 서스펜스/미스터리 전용 (✅ #1·#3·#6 교차검증)

**수수께끼 설계 3요소 — 반드시 갖출 것:**
- 장소: 공간 자체에 서스펜스 기능 (밀실·병원·범죄 현장)
- 인물: 다중 정체성 보유자 + 불신뢰 서술자 최소 1인
- 단서: 소품·언어·행동 단서가 해미 과정에서 기능

**정보 비대칭 유지 (✅ #6·#11):** 관객 정보량 < 인물 정보량 유지. **#11 추가 확인(점진 정보 공개).**

**수수께끼 해소 계단식 상승 (✅ #6):** 해결 직후 더 큰 갈등 등장 → 서스펜스 강도 누적 상승

**서스펜스 서사 구조 4유형 (✅ #6):**
1. 연쇄 고리 구조: 고리당 단일 미션, 고리 간 연결 필수
2. 주부교직 쌍선: 보조선이 주선 전환점 강화
3. 환형·분기 경로: 오프닝↔엔딩 수미상관, 인물 선택이 분기 유발
4. 1+N식: 보조선(N)은 주선(1) 서포트 필수

### 💕 로맨스/여성향 전용

**황금 3초 현수막 선치 (✅ #3·#4·#10 — #10 승급):**
오프닝 15초 내 클라이맥스 먼저 → 역으로 배경 보충. 서스펜스 장치 밀도: 3분 내 6개(=5초 1개)

**쾌감 필수 요소 (✅ #4·#8·#9·#10·#13 — 5편 교차):** 회귀·환생·벼락부자·통쾌한 반격·역전 중 1개 이상

**다단 반전 구조 (✅ #1·#10·#12 — 완전 승급):**
사건 반전 + 인물 성격·감정 방향 반전 병행. 화당 예측 어긋나기 ≥1회

**자기 응시 장면 (🔶 #11·#12 — 그룹 내 2편):** 인물 각성 전환 전 내면 성찰 필수

---

### 🏆 역전/성장극 전용

**핵심 구조 공식 (✅ #4·#13 — 2편):**
- 0~15초 내 갈등 폭발 + 위기→반전 모듈 화당 최소 1사이클
- 극적 대비 주인공: 소인물 시작 → 역전 낙차 극대화

**쾌감 체크리스트 (✅ #4·#8·#9·#10·#13 — 5편):** 회귀·환생·벼락부자·통쾌한 반격·역전 중 1개 이상

**성장선 필수 (✅ #4·#6·#13):** 핵심 실패 최소 3번 → 점층 성장. 환경과의 상호작용으로 표현

**지옥 난이도 초기 과제 (🔶 #8):** 1~2화 내 "거의 불가능한 조건"의 과제 부여. 독립 기준

---

### 👩 도시여성혼인극 전용

**3영역 갈등 교차 필수 (✅ #2·#5 원칙 적용, 🔶 #11 장르 특화):**
①결혼(배신·재산 갈등) ②직장(성차별·재취업) ③가정(고부·육아 불평등) — 단일 영역만 = 주제 빈약

**여성 역전 결말 (🔶 #11·#12 — 그룹 내 2편):**
법적·경제적·전문성 기반 반격. 화해·용서 결말 = 쾌감 소멸. 여성 조력자 장면 ≥1개 필수

**실패 패턴 (도시여성혼인극 특화):**
- 형식적 여성서사: 외형만 여성 중심, 실제는 남성 인정·외모 종속 서사 ⛔ **#11·#12·#13 — 3편 교차 (신규 ✅)**
- 여성 자력 구제 미흡: 남성 조력자·우연에 의존한 해결 ⛔ 🔶 #12

---

### 🕊️ 치유형/생활감형 전용 (🔶 #15)

- 과장 연출 배제: 일상 세절로 감정 로직 구축. 과장된 사건 지양
- 소품 서사 기능화: 회차별 맥락 변화로 관계 발전 암시 (✅ #1 소품 복선과 정합)
- 시각 기호로 가치관 전달 (대사 직접 설교 금지 — ✅ #4·#14와 정합)
- 세부 고증: 소품·장소·시대 세부 정확성

---

### 🇨🇳 주선율 전용 (🔶 #14)

- 가치관 = 대사 선언 금지, 인물 **행동·선택**으로 체현 (✅ #4 설교식 발화 금지와 정합 → **원칙 공유 확인**)
- 결함 있는 인물이 성장 과정에서 가치 실현 → 완벽 인물보다 공감도 높음
