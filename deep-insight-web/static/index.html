Deep Insight

리포팅 등의 다양한 에이전트 응용 개발 위한 커스텀 에이전트 구축 플랫폼

Built on Strands Agent & Amazon Bedrock AgentCore

안전한 오픈소스 아키텍처와 코드를 통해 나만의 에이전트를 구축하세요

📖 사용 가이드

▾
🌐 Chrome 브라우저 사용을 권장합니다
🔒 업로드된 데이터와 생성된 보고서는 고객의 S3 버킷에 저장되며, 프론트엔드는 사내 IP 대역으로 제한되고 백엔드는 프라이빗 VPC에서 운영됩니다. 데이터는 안전하게 관리됩니다.
  1. 📂데이터 파일(CSV, Excel 등)과 컬럼 정의 파일을 업로드합니다
  2. ✏️분석 프롬프트를 입력합니다
  3. 📋분석 계획이 표시되면 승인하거나 수정을 요청합니다
  4. ⏳에이전트가 분석을 완료할 때까지 기다립니다
  5. 📥Reports 섹션에서 생성된 보고서를 다운로드합니다
💡 컬럼 정의 파일은 선택사항이지만 권장합니다. 컬럼명이 도메인 고유 의미를 가질 수 있으며(예: 매출 = 순매출 vs 총매출), 분석에 적용할 수식도 정의할 수 있습니다.
⏱ 소요 시간은 프롬프트 복잡도, 데이터 크기, LLM 서버 상태에 따라 달라집니다. 샘플 데이터 기준 약 15~30분 소요될 수 있습니다.
